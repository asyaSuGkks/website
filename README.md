# Asya Su Gökkuş — Personal Portfolio

A clean, modern, fully static personal portfolio website built with **HTML**, **CSS**, and **vanilla JavaScript**. No frameworks, no build steps — ready to deploy on GitHub Pages.

## 🗂 File Structure

```
├── index.html    # Main page
├── style.css     # All styles (CSS variables, responsive design)
├── script.js     # Interactivity (smooth scroll, animations, mobile menu)
├── assets/       # (Optional) Images, icons, or other static assets
└── README.md     # This file
```

## 🚀 Deploy on GitHub Pages

1. **Create a new GitHub repository** (e.g., `portfolio` or `asyasu.github.io` for a user site).
2. **Upload all files** — `index.html`, `style.css`, `script.js`, the `assets/` folder (if used), and `README.md` — to the root of the repository.
3. Go to your repository **Settings → Pages**.
4. Under **Source**, select **Deploy from a branch**.
5. Choose the **`main`** branch and **`/ (root)`** folder. Click **Save**.
6. Wait a minute or two, then visit the published URL shown on the Pages settings page.

That's it — your portfolio is live! ✨

## 🖥 Run Locally

Simply open `index.html` in any modern browser. No server required.

```bash
# Or use a simple local server (optional):
npx serve .
```

## ✏️ Customization

- **Colors**: Edit the CSS custom properties in `:root` at the top of `style.css`.
- **Content**: Edit the HTML sections directly in `index.html`.
- **Fonts**: The site loads Inter, Outfit, and Fira Code from Google Fonts.
